Welcome to Astro Nightwalk! This guide will help you get started with our beautiful Three.js night city walkthrough theme.

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

## Configuration

Edit `src/config/nightwalk.config.ts` to customize the cityscape. You can adjust:

- Building count and dimensions
- Lighting colors and intensity
- Sky gradient colors
- Camera movement speed
