## Project Overview

A comprehensive analytics dashboard designed for portfolio tracking. Built with React and D3.js, it provides real-time data visualization with smooth animations and interactions.

## Key Features

- **Real-time Updates**: WebSocket integration for live data
- **Interactive Charts**: Line, bar, pie, and custom visualizations
- **Responsive Design**: Optimized for desktop and mobile
- **Dark Mode**: Full dark mode support with smooth transitions

## Technical Stack

- React 18
- D3.js
- TypeScript
- Tailwind CSS
- WebSocket API

## Implementation Details

The dashboard uses a modular chart system that allows for easy addition of new visualization types. Each chart component is self-contained with its own data fetching and rendering logic.
